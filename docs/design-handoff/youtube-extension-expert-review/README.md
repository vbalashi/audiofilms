# YouTube Extension Expert Review Package

This folder is the handoff bundle source for reviewing the AudioFilms YouTube shadowing extension.

Start here:

1. `REVIEW_BRIEF_RU.md` - Russian review brief with product intent, current fragility, architecture questions, and research pointers.
2. `RELATED_EXTENSION_RESEARCH_RU.md` - targeted notes from SubtideX, EasySubs, multi-subs-yt, Language Reactor, Migaku, and YouTube captions docs.
3. `extensions/youtube-shadowing/` in the zip - current unpacked Chrome extension source.
4. `screenshots/SCR-20260610-tmrk-2.png` - current YouTube extension UI for critique.
5. `screenshots/SCR-20260610-tnoq-2.png` - current main AudioFilms web app UI for comparison.
6. `screenshots/SCR-20260611-cdp-clean-wide.png` - current compact extension UI in a clean Chrome profile, wide viewport.
7. `screenshots/SCR-20260611-cdp-clean-narrow.png` - current compact extension UI in a clean Chrome profile, narrow viewport.
8. `docs/intent/subtitle-retrieval-strategy.md` - notes on subtitle provider reliability and fallback options.
9. `docs/intent/subtitle-practice-contract.md` - current app/extension subtitle metadata and phrase contract.
10. `docs/intent/youtube-extension-validation-matrix.md` - validation fixtures, checks, and latest smoke results.
11. Selected main AudioFilms app files in the zip - for comparison with the extension approach.

The zip archive is generated at:

```text
docs/design-handoff/youtube-extension-expert-review/audiofilms-youtube-extension-expert-review.zip
```
